#  listClusters 
Returns , in a table format, a list of clusters.  

The table shows the cluster name, number of nodes and the services

Example
```
Clusters
------------------------------------------------------------------------------
|       Name       |                  ID                  | Nodes | Services |
------------------------------------------------------------------------------
| Cluster01        | 0887f5d9-f7ca-4a4d-9486-727ae95ca88d | 3     | ['data'] |
------------------------------------------------------------------------------

```
